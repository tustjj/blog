# 简介

<TOCInline toc={props.toc} exclude="简介" toHeading={2}/>

[Anaconda](https://www.anaconda.com/)是一个用于科学计算的[Python](https://www.python.org/)发行版，支持Linux, Mac, Windows,
包含了众多流行的科学计算、数据分析的Python包。安装了Anaconda，你就拥有了任意版本的python和各种包了。 使用Anaconda，在同一个系统上维护不同的项目环境也很方便。

## 常用命令

- 查看conda版本
```bash
$ conda --version
```

- 查看已安装哪些依赖库
```bash
$ conda list
```

- 创建新的python环境
```bash
$ conda create --name myenv python==3.7
```

- 复制环境
```bash
$ conda create --name myclone --clone myenv
```

- 查看所有python环境
```bash
$ conda info --envs
$ conda env list
```

- 激活、进入某个python环境
```bash
$ source activate myenv
```

- 退出环境
```bash
$ source deactivate
```

- 删除环境
```bash
$ conda remove --name myenv --all
```
- 查看某个环境下安装的库
```bash
$ conda list -n myenv
```

- 查找包
```bash
$ conda search xxx
```

- 安装包
```bash
$ conda install xxx
```

- 删除包
```bash
$ conda remove xxx
```

- 安装到指定环境
```bash
$ conda install -n myenv xxx
```
