# 简介

<TOCInline toc={props.toc} exclude="简介" toHeading={2}/>

[SQL](https://en.wikipedia.org/wiki/SQL)是一种特定的查询语言，向DBMS发送查询请求，查询结构化数据库，DBMS返回查询结果。

## 常用命令

- 创建数据库/表/索引
```bash
CREATE DATABASE life

CREATE TABLE today
(   id char(13) PRIMARY KEY,
    name varchar(10) NOT NULL UNIQUE,
    major varchar(30) NULL CHECK(major IN('基础课', '专业课', '选修课')),
    age int DEFAULT 0,
    birthday date NULL
);

CREATE INDEX
```

- 删除数据库/表/索引
```bash
DROOP DATABASE life
```
